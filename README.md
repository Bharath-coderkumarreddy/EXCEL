# Data Transformation and Data Modeling using Power BI

## Project Overview
This project demonstrates the process of data transformation and data modeling in Power BI. The objective is to clean, transform, and organize raw data into a structured model that can be used for reporting, visualization, and business analysis.

## Features
- Data cleaning and preprocessing
- Data transformation using Power Query
- Data modeling with relationships
- Creation of calculated columns and measures
- Optimized data structure for reporting
- Interactive Power BI report

## Tools Used
- Microsoft Power BI Desktop
- Power Query Editor
- Data Modeling

## Project Structure
```
Data Transformation and Data Modeling.pbix
README.md
```

## Data Transformation Steps
1. Imported raw datasets into Power BI.
2. Removed unnecessary columns.
3. Renamed columns for better readability.
4. Handled missing or null values.
5. Changed data types where required.
6. Merged and transformed tables using Power Query.
7. Created aggregated and summarized datasets.

## Data Modeling
- Established relationships between tables.
- Configured primary and foreign key relationships.
- Implemented a star schema approach where applicable.
- Ensured data integrity and optimized performance.

## Key Learnings
- Power Query transformations
- Data cleaning techniques
- Relationship management
- Data model optimization
- Business intelligence reporting

## How to Open the Project
1. Download the repository.
2. Open `Data Transformation and Data Modeling.pbix` in Power BI Desktop.
3. Refresh the data if required.
4. Explore the data model and report pages.
